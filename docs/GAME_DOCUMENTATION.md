# 真理序列 · 测试版功能文档

> 知识逻辑卡牌 Roguelike 游戏

---

## 一、游戏概览

玩家使用「知识点」卡牌组成连携（Combo），在真理合成台上验证逻辑，对 BOSS 造成伤害。正确组合造成高额伤害与护盾，错误组合会触发逻辑悖论反噬。每局战斗随机遇到不同 BOSS，各有固定行动模式（攻击 / 防御 / 弃牌 / 诅咒）。

---

## 二、界面与流程

### 2.1 首页 (HomeView)

| 功能 | 状态 | 说明 |
|------|------|------|
| 开启课题 | 敬请期待 | 点击提示「敬请期待」 |
| 真理图鉴 | 敬请期待 | 未开放 |
| **测试模式** | ✅ 可用 | 展开后选择学科，直接进入战斗 |
| 系统设置 | 敬请期待 | 未开放 |

**测试模式学科选项**：
- 所有（使用全部可连携卡牌）
- 地理
- 生物
- 历史

### 2.2 地图 (MapView)

在正式流程中，玩家在此选择路径，进入不同节点：
- **测验**：进入战斗
- **自习室**：进入营地休整
- **随机事件**：进入随机事件

每层显示 3 个节点，顶部显示层数、遗物、HP。

### 2.3 战斗 (BattleView)

核心玩法界面，详见下一节。

### 2.4 卡牌选择 (CardChoiceView)

战斗胜利后出现（非 BOSS 战）：从 3 张随机卡牌中选择 1 张加入卡组。

### 2.5 遗物奖励 (RelicRewardView)

击败 BOSS 后出现：从 3 件遗物中选择 1 件。若已集齐全部遗物，可直接继续。

### 2.6 自习室 (CampView)

休整节点，三选一：
- **小憩**：回复 30% 最大 HP
- **温故知新**：升级一张卡牌（每张卡只能升级一次）
- **知识迭代**：遗忘一张卡牌（卡组至少保留 1 张）

### 2.7 随机事件 (EventView)

随机抽取一个事件，展示标题、描述，提供多个选项。每个选项有不同效果（扣血、回复、获得/删除卡牌等）。

## 三、战斗系统详解

### 3.1 基本流程

1. 从手牌区点击卡牌，放入真理合成台（最多 4 张）
2. 点击「验证逻辑」进行判定
3. **无论成功或失败**，BOSS 都会执行一次行动（攻击 / 防御 / 弃牌 / 诅咒）
4. 若 BOSS 攻击，会弹出伤害提示
5. 手牌重新抽取 8 张（含抽牌堆与弃牌堆）

### 3.2 真理合成台

- 最多放入 **4 张**卡牌
- 放入的卡可点击移回手牌
- 验证后合成台上的卡进入弃牌堆，不再参与本次战斗

### 3.3 验证结果

| 情况 | 效果 |
|------|------|
| 正确连携（2–4 张） | 造成 Combo 伤害，可能获得护盾，显示知识点与事实文本 |
| 单卡验证 | 造成 2–5 点弱伤害 |
| 错误组合 | 触发逻辑悖论，受到 `张数 × 5` 点反噬伤害（有遗物可减半） |

### 3.4 BOSS 行为

右上角显示：
- **攻击力**：下次攻击的伤害值
- **防御**：当前护盾值

BOSS 行动类型：

| 类型 | 效果 |
|------|------|
| attack | 对玩家造成指定伤害 |
| defend | 为自己增加护盾 |
| discard | 随机弃掉玩家若干手牌 |
| curse | 将一张诅咒卡加入抽牌堆（如「久坐血栓」「白日梦」等） |

玩家造成的伤害优先扣除 BOSS 护盾，溢出部分才扣 BOSS HP。

### 3.5 战斗内功能

| 功能 | 说明 |
|------|------|
| 验证逻辑 | 判定合成台上的卡牌 |
| 整理思绪 | 丢弃全部手牌并重抽 8 张，每场战斗限 1 次 |
| 玩法帮助 | 查看玩法说明及与本局牌组相关的 Combo 配方 |
| 连携提示 (3) | 每场 3 次，高亮可形成连携的手牌 |

### 3.6 胜负条件

- **胜利**：BOSS HP ≤ 0
  - 普通敌人 → 进入卡牌选择
  - BOSS → 进入遗物奖励
- **失败**：玩家 HP ≤ 0 → 显示「战斗失败」弹窗，可返回首页

## 四、卡牌与连携

### 4.1 卡牌结构

每张卡包含：`id`、`name`、`subject`（学科）、`type`、`desc`。学科包括：地理、生物、历史。

### 4.2 连携 (Combo)

- 需 2–4 张卡按固定组合才能触发
- 每个连携有：`requires`（所需卡牌 ID）、`damage`、`block`、`name`、`factText`

### 4.3 遗物影响

| 遗物 | 效果 |
|------|------|
| 郑和的罗盘 | 所有地理 Combo 伤害 +30% |
| 显微镜 | 2 卡连击伤害翻倍 |
| 破损的怀表 | 悖论反噬伤害减半 |

### 4.4 特殊卡牌

- **curse_celestial_dream**（白日梦）：抽到即受到 2 点伤害
- 其他诅咒卡会由 BOSS 的 curse 行为加入牌堆

## 六、玩家状态

- **HP**：初始 100，归零即战斗失败
- **护盾**：可吸收伤害，每回合末清空（敌人回合结束时）
- **卡组**：10 张起手，战斗胜利可加卡，营地可遗忘
- **遗物**：最多 3 件，击败 BOSS 获得

## 七、UI 说明

### 7.1 战斗界面

- **顶部左侧**：遗物、玩家 HP、护盾、玩法帮助、连携提示
- **顶部右侧**：BOSS 攻击力、护盾、头像、名称、HP 条（血条加粗，文字在血条上显示）
- **中部**：真理合成台与 4 个卡槽
- **底部**：手牌区、抽牌堆 / 弃牌堆数量

### 7.2 伤害与提示

- **BOSS 伤害提示**：屏幕顶部居中，2.5 秒后消失
- **连携提示 Toast**：屏幕底部居中
- **战斗失败**：全屏弹窗，需点击「返回首页」
